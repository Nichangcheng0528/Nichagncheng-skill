# Skill Generator - Skill 生成器

本 Skill 提供完整的 Skill 生成流程，包括文件创建、质量检测、规范性打分、自动修复等全链路闭环。

## 触发条件

当用户提到以下任何内容时立即使用本 Skill：
- 创建 skill
- 新建 skill
- 生成 skill
- 封装新技能
- 制作 skill
- 开发 skill
- skill生成

## 核心功能

### 1. 自动文件创建
- 智能生成 SKILL.md 模板
- 自动创建目录结构
- 支持多种 Skill 类型

### 2. 质量检测（5 大维度）

| 维度 | 检测项 | 分值 |
|------|--------|------|
| **文件结构** | 目录存在、文件完整、路径正确 | 20分 |
| **编码检测** | UTF-8 编码、无 BOM、无乱码、无空字符 | 20分 |
| **格式规范** | frontmatter 完整、字段齐全、格式正确 | 20分 |
| **内容质量** | 描述清晰、触发条件明确、文档完整 | 20分 |
| **可调用性** | 语法正确、逻辑通顺、示例可用 | 20分 |

### 3. 规范性打分系统

**评分标准：**
- 90-100 分：优秀，可直接使用
- 70-89 分：良好，需小幅优化
- 50-69 分：一般，需要较多改进
- 50 分以下：不合格，需重新生成

### 4. 自动修复
检测到问题时自动修复：
- 编码问题：重新编码为 UTF-8 无 BOM
- 格式问题：补全 frontmatter
- 内容问题：补充必要字段

### 5. 配置注册
- 自动添加到 skill-config.json
- 生成触发关键词
- 更新触发规则（可选）

## 完整工作流程

```
用户请求 → 确认需求 → 生成模板 → 质量检测
    ↓
  打分评估
    ↓
分值 ≥ 90? ──是──→ 注册到配置 → 完成
    ↓ 否
  自动修复 → 重新检测
    ↓
  分值仍低 → 提示用户
```

## 分类体系

技能按以下分类体系组织：
- business/: 业务类技能（CRM、销售、数据分析等）
- productivity/: 生产力工具（文档处理、周报生成等）
- audio/: 音频处理（语音转写、录音分析等）
- meta/: 元技能（技能生成、上传、管理等）
- platform/: 平台集成（第三方服务对接等）

## 注意事项

1. 生成后必须执行质量检测
2. 分值低于 90 分需优化
3. 上传前必须脱敏检查
4. 保持触发条件简洁明确
