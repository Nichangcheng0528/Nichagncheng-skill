# Skill Generator - Skill 生成器

Skill 生成器，用于创建新的 Skill 并进行质量检测。

## 功能特性

- 自动文件创建：智能生成 SKILL.md 模板
- 质量检测：5 大维度 100 分制评分系统
- 自动修复：编码问题、格式问题、内容问题自动修复
- 配置注册：自动添加到 skill-config.json
- 迭代优化：生成后自动检测、自动修复、自动优化

## 使用场景

当用户要求创建 skill、新建 skill、生成 skill 或封装新技能时立即使用

## 触发关键词

- 创建 skill
- 新建 skill
- 生成 skill
- 封装新技能
- 制作 skill

- skill生成

## 质量评分标准

| 维度 | 分值 |
|------|------|
| 文件结构 | 20分 |
| 编码检测 | 20分 |
| 格式规范 | 20分 |
| 内容质量 | 20分 |
| 可调用性 | 20分 |
| **总计** | **100分** |

### 评分等级

- 90-100 分：优秀，可直接使用
- 70-89 分：良好，需小幅优化
- 50-69 分：一般，需要较多改进
- 50 分以下：不合格，需重新生成
