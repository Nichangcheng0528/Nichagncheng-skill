# Skill Uploader - 技能上传助手

自动化技能上传助手，用于检查技能规范性、执行脱敏处理、上传到 GitHub 仓库。

## 功能特性

- 规范性检查：验证 SKILL.md、_meta.json、README.md 等文件完整性
- 脱敏处理：自动扫描并替换敏感信息（API Key、手机号、密码等）
- GitHub 上传：自动初始化 Git、提交代码、推送到远程仓库
- 分类上传：支持按业务领域分类上传到指定目录

## 使用场景

当用户需要上传技能到 GitHub 时立即使用

## 触发关键词

- 上传技能
- 检查技能脱敏
- 批量上传
- 发布技能到GitHub
- skill上传

## 技术规范

- 编码：UTF-8 无 BOM
- 格式：Markdown
- 依赖：GitHub API Token

- 目标仓库：https://github.com/Nichangcheng0528/Nichagncheng-skill
