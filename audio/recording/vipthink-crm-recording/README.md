## 功能概述

本技能封装了从 VIPThink CRM 系统调取电话录音并进行智能分析的完整操作流程：

| 阶段 | 功能 | 技术方案 |
|------|------|---------|
| **录音调取** | 从 CRM 筛选学生 → 获取有效通话录音 URL | Playwright + Chrome CDP |
| **语音转录** | 将录音转为文字（支持普通话、粤语） | 阿里云 DashScope ASR (Paraformer-v2) |
| **智能分析** | 分析对话流程、客户意向、关键话术、改进建议 | 阿里云 DashScope LLM (Qwen-Plus) |
| **报告生成** | 生成多维度分析报告（单条 / 批量） | Python + Markdown |

### 核心能力

- ✅ **CRM 2.0 完整支持**：登录、筛选、录音获取全流程自动化
- ✅ **双语语音识别**：普通话 + 粤语自动检测与转录
- ✅ **说话人分离**：自动区分 CC/TMK 和客户双方对话
- ✅ **多维度报告**：CC 绩效、客户意向、转化评估、管理建议
- ✅ **批量分析**：支持一次分析多条录音，生成汇总报告

## 快速开始

### 前置条件

| 依赖 | 说明 |
|------|------|
| Python 3.8+ | 运行分析脚本 |
| Google Chrome | CRM 自动化操作（需启用调试端口） |
| Playwright | `pip install playwright && python -m playwright install chromium` |
| 阿里云 DashScope API Key | 用于 ASR 转录和 AI 分析 |
| CRM 2.0 账号 | 登录 ecc.vipthink.cn |

### 配置

在使用前，需要配置以下敏感信息（**请勿将真实凭证提交到代码仓库**）：

```bash
# 方式1：环境变量（推荐）
export CRM_USERNAME="你的CRM账号"
export CRM_PASSWORD="你的CRM密码"
export DASHSCOPE_API_KEY="你的DashScope API Key"

# 方式2：.env 文件（添加到 .gitignore）
CRM_USERNAME=你的CRM账号
CRM_PASSWORD=你的CRM密码
DASHSCOPE_API_KEY=你的DashScope API Key
```

### API 配置

| 配置项 | 说明 |
|--------|------|
| ASR 模型 | `paraformer-v2`（支持普通话、粤语） |
| ASR 端点 | `https://dashscope.aliyuncs.com/api/v1/services/audio/asr/transcription` |
| 分析模型 | `qwen-plus` |
| 分析端点 | `https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions` |

## 报告模块说明

录音分析报告包含 **6 大模块**：

| 模块 | 内容 | 关键指标 |
|------|------|---------|
| **一、当日录音总览** | 核心指标汇总、意向分布、时长分布 | 有效通话率、高/中/低意向占比 |
| **二、CC 维度分析** | CC 绩效汇总表、梯队划分（ABC 三档） | 综合评分（4 星制）、平均时长 |
| **三、逐条录音分析** | 每条录音详细信息 + 转化评估 | 转化评估标签 |
| **四、转化评估汇总** | 按转化评估分类、优先追单名单 | P0/P1 优先级 |
| **五、团队管理与培训建议** | 团队诊断、经理建议、主管个性化建议 | 问题严重程度矩阵 |
| **六、技术说明与下步计划** | 数据来源、已知问题、修复方案 | — |

### CC 综合评分维度

| 梯队 | 评分 | 特征 |
|------|------|------|
| A 梯队 | ⭐⭐⭐⭐ | 高意向转化，流程闭环能力强 |
| B 梯队 | ⭐⭐⭐ | 流程规范，需强化促单 |
| C 梯队 | ⭐⭐ | 需重点培训，通话时长不足或话术模板化 |

### 转化评估 4 级分类

| 等级 | 标签 | 判断标准 |
|------|------|---------|
| 可追单转化 | 已安排试听/添加微信 | 24-48h 内可推进 |
| 需培育跟进 | 有初步兴趣但存在顾虑 | 需持续培育 |
| 待激活 | 意向不明或通话质量低 | 需重新激活 |
| 暂不推荐 | 明确拒绝或无响应 | 短期不追单 |

## 故障排除

### CRM 操作类

| 问题 | 解决方案 |
|------|---------|
| browser_evaluate 返回 null | 检查 preload 脚本；刷新页面后重试 |
| 弹窗未出现 | 等待 2 秒后检查；确认点击正确元素 |
| 日期选择后数据为空 | 使用日历面板点击法（单日点两次） |
| 只获取到第一页数据 | 滚动到底部 → 等待分页渲染 → 遍历所有页 |
| 通话记录显示 0 条 | CRM 改版后不在 pane 内，直接在弹窗中查找 table |

### CDP/Chrome 连接类

| 问题 | 解决方案 |
|------|---------|
| `connect ECONNREFUSED` | 用 `Start-Process` 启动 Chrome 并指定 `--remote-debugging-port=9222` |
| API 返回 1003 未登录 | 在 CDP 连接的 Chrome 中重新登录 CRM |
| Playwright launch 和 CDP connect 登录不共享 | 统一使用 `launch_persistent_context()` |

### 语音分析类

| 问题 | 解决方案 |
|------|---------|
| ASR 转录失败 | 检查录音 URL 是否过期（OSS 签名 7 天有效） |
| 粤语识别不准确 | 确认 `language_hints` 包含 `yue` |
| API Key 无效 | 确认使用 DashScope API Key（`sk-` 开头） |
