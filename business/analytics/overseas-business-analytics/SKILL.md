## 五、业务背景速查

### 5.1 组织架构（四级）

```
总监（用户）
  └── 区域团队经理
          └── TL（Team Leader）
                  └── CC（一线销售）
```

### 5.2 岗位职责

| 岗位 | 核心职责 | 相关模块 |
|------|---------|---------|
| TMK | 邀约：摸排数据 → 过滤需求 → 约课 | 03-sales, 07-data-bi |
| CC | 关单：到课通知 → 完课转化 → 移交LP | 03-sales, 04-delivery |
| LP | 班主任：后端服务、续费管理 | 05-learning-partner |

### 5.3 业务流程

```
投放获客 → TMK摸排约课 → [线索分发] → CC沟通(通时通次)
                                              ↓
                              [流转管理](池子/配额/公海)
                                              ↓
                              CC约课(约课行为监控)
                                              ↓
                              到课(应到课监控/到课通知)
                                              ↓
                              Demo课(体验课/满班率)
                                              ↓
                              课后跟进(SOP/T+N)
                                              ↓
                              关单转化 → GMV
                                              ↓
                              移交LP(长线服务)
```

## 九、分析场景与BI操作流程

### 9.1 场景1：新签业绩分析

**触发**：GMV、业绩达成、本月完成

**BI操作流程**：
1. 登录BI → 益智海外分团队业绩达成_月度
2. 设置筛选器：日期=本月、团队=全部
3. 提取数据：GMV目标、GMV达成、净GMV、退费率
4. 导出数据或截图

**筛选器配置**：
```json
{
  "本月开始日期": "2026-05-01",
  "本月结束日期": "2026-05-17",
  "主投": "全部",
  "渠道一级分类": "全部"
}
```

### 9.2 场景2：CC过程管理分析

**触发**：CC转化、通时通次、约客量

**BI操作流程**：
1. 登录BI → 益智CC日通时通次监控
2. 设置筛选器：日期=近7天、CC姓名=全部
3. 提取数据：通时、通次、约客量、完客量、到课率
4. 按CC排名分析

**筛选器配置**：
```json
{
  "日期": "2026-05-17",
  "CC姓名": "全部"
}
```

### 9.3 场景3：TMK跟进分析

**触发**：TMK表现、邀约率、到课率

**BI操作流程**：
1. 登录BI → 个人维度-海外TMK跟进情况
2. 设置筛选器：本月开始日期、本月结束日期
3. 对比本月vs上月数据
4. 分析邀约率、出席率、完课率变化

**筛选器配置**：
```json
{
  "本月开始日期": "2026-05-01",
  "本月结束日期": "2026-05-17",
  "上月开始日期": "2026-04-01",
  "上月结束日期": "2026-04-30"
}
```

### 9.4 场景4：满班率分析

**触发**：满班率、Demo课效率

**BI操作流程**：
